# GitHub Repository


Command	Purpose
- git branch	Shows all local branches
- git branch -vv	Shows local branches and their tracking information
- git remote -v	Shows connected remote repositories
- git remote show origin	Shows detailed information about the remote repository
- git status	Shows repository status and branch tracking information
